Create user avatars using their initial letters.


== Description ==

The Leafio Letter Avatar plugin allows you to create stylish user avatars using just their initial letters. Built entirely with CSS, it will not increase your hosting or server storage and has no impact on your site's speed or performance.


== Installation ==
1. Upload [`leafio-letter-avatar`] to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
